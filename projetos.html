HTML5 CSS3 JavaScript Bootstrap API

Funcionalidades Principais

  • Catálogo dinâmico integrado com Open Library API

  • Checkout com validação de formulários
  • Múltiplas opções de pagamento (Cartão, PIX, Boleto)
  • Sistema de temas (Vermelho, Roxo, Azul, Multicolor)

  • Carrossel de destaques e produtos populares
